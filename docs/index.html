MeetAll Dataset

MeetAll comprises 231 meetings, 200 h audio, bilingual transcripts, and 2 000 agent QA turns with high-fidelity voice cloning.

  • 108 complexity cells across 4 cognitive axes
  • Voice-cloned agent utterances via F5-TTS
  • CC BY-NC 4.0 license
MeetBench Benchmark

MeetBench adapts CompassJudger and Prometheus to quantify assistant responses along five dimensions plus overall Prometheus score.

Scripts located in /MeetBench_Benchmark/.

MeetMaster Agent

MeetMaster employs parallel Planner and Talker modules, reducing latency by up to 1.5 s while boosting quality by 22.5 % on complex queries.

Implementation in /baseline/MeetMaster/.
